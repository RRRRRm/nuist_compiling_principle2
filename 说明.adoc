= 说明

= 行首语法

* 标题 `=空格`
* 无序列表 `*空格`
* 段落分割 `空行`
* 分割线 `\***`

= 行内语法 =

* 粗体 用 `*` 包裹, 有空格
* 斜体 用 `_` 包裹, 有空格, 粗体斜体可以嵌套
* 下划线 `[.underline]\#内容#`
* 删除线 方括号里换成 `.line-through`
* 大小 方括号 `.small` `.big`
* 上下标 `^` `~` 包裹
